## 🧪 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML 5
- CSS 3

## 💻 Projeto

A ideia do projeto é simular um celular acessando as redes sociais mais usadas geralmente e simular quando acessa essas redes.

## 📖 O que eu aprendi
